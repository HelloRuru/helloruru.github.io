AIO VIEW 操作手冊

    
監測你的文章是否出現在 Google AI Overview 搜尋摘要。
從 Sitemap 解析到報告分析，一站完成。

    

  


  
  

    

      

        
      

      
OPERATION FLOW

    


    

      

        
01

        

          
貼 Sitemap + 選文章

          
輸入你網站的 sitemap.xml 網址，按「解析」。系統自動擷取所有文章，並用規則引擎自動產生搜尋語句（不需 API）。

          用年份、月份篩選，勾選你想監測的項目。語句也可以手動編輯。

        

      


      

        
02

        

          
檢查 AIO

          
兩種方式檢查你的文章是否出現在 Google AI Overview：

          手動檢查（免安裝） — 按「開始手動檢查」，系統會逐篇開啟 Google 搜尋，你在搜尋結果頁面點「有 AIO / 沒有 AIO」回報。也可以拖曳書籤列按鈕到瀏覽器書籤列，之後在任何 Google 搜尋頁面一鍵回報。

          CLI 自動掃描（進階） — 產生 queries.json 後，在電腦跑 node scan.js，每篇間隔 2.5 分鐘，適合睡前放著跑。

        

      


      

        
03

        

          
看報告

          
手動檢查的結果即時顯示在報告區。CLI 掃描完成後拖曳 results.json 到上傳區。

          自動產生圖表分析（AIO 分布、引用來源排行）、時間軸報告（歷史趨勢、前後對比），支援 CSV 匯出。

        

      

    

  


  
  

    

      

        
      

      
FIRST-TIME SETUP

    


    
手動檢查模式不需要安裝任何東西，直接在瀏覽器操作即可。
以下安裝只有使用 CLI 自動掃描（進階模式）才需要，而且只需做一次。


    

      

        
        
        
        PowerShell
      

      

# 1. 裝 Node.js — 到 nodejs.org 下載 LTS 版，一直按下一步
# 2. 裝 Git — 到 git-scm.com/download/win 下載，一直按下一步
# 3. 下載工具 + 安裝依賴

git clone https://github.com/helloruru/helloruru.github.io.git
cd helloruru.github.io/aio-view/cli
npm install
npx playwright install chromium

# 完成！可以開始掃描了
node scan.js --input queries.json --output results.json --domain 你的網域.com
      

    

  


  
  

    

      

        
      

      
NEW FEATURES

      v2.0
    


    

      

        AI ASSIST
        
AI 輔助面板

        
超過 15 篇自動分批。複製提示詞給 AI → 把回覆貼回來 → 系統用模糊比對自動配對標題和語句。

      


      

        CHARTS
        
圖表分析

        
AIO 狀態甜甜圈圖（被引用 / 有 AIO 未引用 / 無 AIO），加上引用來源 Top 10 柱狀圖。

      


      

        TIMELINE
        
時間軸報告

        
趨勢折線圖追蹤引用率變化。歷史記錄卡片、兩次掃描對比（新增/失去/持續引用）。

      


      

        CLI
        
AI 語句產生器

        
用 API Key 自動產生搜尋語句。支援 OpenAI、Anthropic。50 篇約 NT$1-3。

      

    


    

      

        
        
        
        AI Generate
      

      

# OpenAI（GPT-4o-mini，最便宜）
node ai-generate.js --input queries.json --provider openai --model gpt-4o-mini

# Anthropic（Claude Haiku，最快）
node ai-generate.js --input queries.json --provider anthropic --model claude-haiku-4-5-20251001
      

    

  


  
  

    

      

        
      

      
INTEL NOTES

    


    

      

        >
        
每篇間隔 2.5 分鐘，50 篇約 2 小時。適合睡前放著跑，起床就有報告。

      

      

        >
        
中途可以關掉，下次會接著上次的進度繼續，不會遺失已完成的結果。

      

      

        >
        
歷史記錄存在瀏覽器 IndexedDB，關閉分頁、重開瀏覽器都不會消失。

      

      

        >
        
AI 語句產生費用參考：50 篇約 NT$1-3（GPT-4o-mini）或 NT$0.5（Claude Haiku）。

      

      

        >
        
掃描時不要另開 Chrome 視窗，讓 Playwright 用獨立瀏覽器跑就好。
