# AIO View Chrome 擴充功能

零成本的 Chrome 未封裝擴充功能，專門在 Google 搜尋頁自動偵測 AI Overview，並把結果回傳給 AIO View。

## 安裝

1. 開啟 Chrome 的 `chrome://extensions`
2. 打開右上角「開發人員模式」
3. 按「載入未封裝項目」
4. 選擇這個資料夾：`aio-view/chrome-extension`

## 使用

1. 先在 AIO View 選好文章
2. 點「開始自動檢查」
3. 擴充功能會在 Google 搜尋頁自動抓取 AIO 狀態
4. 結果會自動回傳到 AIO View

## 說明

- 不需要 API key
- 不需要付費服務
- 不需要油猴
- 只要 Chrome 與這個 repo 就能用
