AIO View

監測你的文章是否出現在 Google AI Overview

AIO View 幫你查看網站文章有沒有出現在 Google AI Overview（AI 摘要）裡。

需要在電腦跑一個小工具來搜尋 Google，第一次要先裝環境。

第一次使用？先裝這些

  1. 裝 Node.js

    到 nodejs.org 下載 LTS 版，一直按下一步。

  2. 裝 Git

    到 git-scm.com 下載，一直按下一步。

  3. 下載工具

    打開 PowerShell，貼上：

    git clone https://github.com/helloruru/helloruru.github.io.git cd helloruru.github.io/aio-view/cli
    npm install
    npx playwright install chromium

怎麼用

  1. 貼上 Sitemap 網址，按「解析」

    如果是 Sitemap Index（WordPress 常見），會出現下拉選單讓你選子 Sitemap。

  2. 挑選要監測的文章

    用年份、月份篩選，勾選想監測的文章。

  3. 填寫搜尋語句

    每篇文章需要一個搜尋語句（模擬使用者會怎麼搜）。可以手動編輯，也可以：

    按「複製 AI 提示詞」→ 貼到 ChatGPT 或 Claude → 拿到語句 → 按「批次貼入語句」一次貼回來。

  4. 產生指令，下載 queries.json

    按「產生 CLI 指令」，把下載的 queries.json 放到 aio-view/cli 資料夾。

  5. 在電腦跑掃描

    在 PowerShell 輸入：

    node scan.js --input queries.json --output results.json --domain 你的網域.com
  6. 上傳 results.json 看報告

    掃描跑完後，把 results.json 拖到下方「上傳結果」區就好。

小提醒

  • 每篇間隔 2.5 分鐘，50 篇約 2 小時，適合睡前放著跑
  • 中途可以關掉，下次會接著上次的進度
  • 跑的時候不要開其他 Chrome 視窗

進階模式：用你的 API Key 自動產生搜尋語句。先匯出搜尋語句 JSON，再用 CLI 工具一鍵產生。

步驟 1：匯出 queries.json（上方「匯出搜尋語句」按鈕）

步驟 2：在 aio-view/cli 資料夾執行：

# 使用 OpenAI（GPT-4o-mini，最便宜） node ai-generate.js --input queries.json --provider openai --model gpt-4o-mini # 或使用 Claude（Haiku，最快） node ai-generate.js --input queries.json --provider anthropic --model claude-haiku-4-5-20251001

步驟 3：產生完的 queries.json 會自動更新語句欄位，直接用來掃描即可。

費用參考：50 篇約 NT$1-3（GPT-4o-mini）或 NT$0.5（Claude Haiku）
