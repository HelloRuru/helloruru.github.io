AIO View

監測你的文章是否出現在 Google AI Overview

AIO View 監測你的網站文章是否被 Google AI Overview（AI 摘要）引用。

由於瀏覽器限制，需要在你的電腦執行小工具來搜尋 Google。

事前準備（只需做一次）

  1. 安裝 Node.js

    前往 nodejs.org 下載安裝，選擇 LTS 版本，一直按下一步即可。

  2. 安裝 Git

    前往 git-scm.com 下載安裝，一直按下一步即可。

  3. 下載 CLI 工具

    開啟 PowerShell（在開始選單搜尋「PowerShell」），輸入：

    git clone https://github.com/helloruru/helloruru.github.io.git

    接著安裝相依套件：

    cd helloruru.github.io/aio-view/cli
    npm install
    npx playwright install chromium

使用流程

  1. 在這個網頁：輸入 Sitemap 網址

    在下方輸入你的網站 sitemap.xml 網址，點「解析」。

  2. 在這個網頁：選擇要監測的文章

    勾選想監測的文章，可編輯搜尋關鍵字。點「產生 CLI 指令」會自動下載 queries.json。

  3. 在電腦：把 queries.json 放到 CLI 資料夾

    把下載的 queries.json 移到 helloruru.github.io/aio-view/cli 資料夾。

  4. 在電腦：執行掃描

    在 PowerShell 輸入（確保在 cli 資料夾）：

    node scan.js --input queries.json --output results.json --domain 你的網域.com

    然後就會看到瀏覽器自動打開、搜尋，慢慢等它跑完。

  5. 在這個網頁：上傳結果

    掃描完成後，在下方的「上傳結果」區塊上傳 results.json，就能看到監測報告！

注意事項

  • 每篇文章間隔 2.5 分鐘搜尋，避免被 Google 封鎖
  • 50 篇文章約需 2 小時，建議睡前或外出時執行
  • 執行中途可以關閉，下次會從上次進度繼續
  • 請勿同時開其他 Chrome 視窗，可能會干擾掃描
