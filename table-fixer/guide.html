表格貼上就亂掉？四個原因與修復方法

Table Fixer 使用指南｜Kaoru Tsai 撰寫・2026-07 更新

從網頁複製一張表格，貼到 Word 的瞬間，欄位散得到處都是。資料還在，只是全部擠錯了位置 — 這不是操作錯誤，是格式在搬家的路上弄丟了。

本指南說明表格亂掉的四個常見原因，以及用 Table Fixer 三步驟還原的方法。

表格為什麼會亂掉

原因常見來源亂掉的樣子
分隔符不一致網頁、Email 轉貼有的欄用 Tab 分隔、有的用空格，貼上後欄位對不齊
全形半形混用中文文件、LINE 訊息全形空格和半形空格寬度不同，看起來對齊、貼上就歪
儲存格內換行PDF 複製一格裡的多行文字被拆成好幾列，一列資料變三列
合併儲存格攤平Excel、報表系統合併格複製後只剩一份值，其他列的欄位整排位移

三步驟修復

  1. 貼上 — 把亂掉的表格文字整段複製，貼進 Table Fixer 輸入區
  2. 按「修復表格」 — 工具判斷欄位分隔並還原成表格，結果直接顯示在下方，欄位對錯一眼看得出來
  3. 選格式複製 — 要貼 Word / Google Docs 選「複製表格」，要貼 Excel 選「複製純文字」
開啟 Table Fixer 修表格

貼回文件的小提醒

  • 「複製表格」貼到 Word / Google Docs：保留表格框線與格式
  • 「複製純文字」貼到 Excel：Tab 分隔的純文字，Excel 會自動分欄
  • 貼完先檢查最後一欄 — 欄位錯位通常從最右邊開始看得出來

常見問題

Q1、為什麼表格從網頁複製到 Word 會亂掉？

複製時只帶走文字，表格結構（欄位邊界）常被轉成 Tab、空格或換行。目的地程式猜不出原本的欄位邏輯，就把資料排錯位置。

Q2、Table Fixer 支援哪些來源的表格？

只要是文字形式的表格都能處理：網頁、PDF、Excel、LINE 或 Email 轉貼都可以。圖片形式的表格要先用 OCR 轉成文字。

Q3、貼上的資料會被儲存嗎？

不會。Table Fixer 全程在瀏覽器端運作，資料不會上傳到任何伺服器。
