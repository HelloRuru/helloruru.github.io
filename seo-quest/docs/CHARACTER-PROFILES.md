### 愛絲（S）伊歐（EO）

#### 📌 基本資料

| 項目 | 內容 |
|------|------|
| **全名** | 愛絲（S）伊歐（EO） |
| **暱稱** | 伊歐、SEO |
| **本質** | SEO 的化身 |
| **角色定位** | 全域導航員 / 宇宙最強領航者 |
| **代表色** | 🌸 玫瑰粉 + 🍷 酒紅 |
| **代表符號** | 🧭 指南針 + 📚 筆記本 + ✨ 銀色長杖 |
| **核心信仰** | 在複雜的規則中，永遠存在一條最優雅的捷徑 |
| **標誌金句** | 「別擔心，讓我帶你找到演算法的最短路徑。」 |

#### 🎯 核心標籤

- `#全域路徑規劃` - 不只看單一頁面，而是能規劃整體的流量航道
- `#白帽文案生成` - 用最不費力、最自然的方式，達成最頂尖的排名
- `#數據定海神針` - 無論搜尋趨勢如何波動，只要有她在，節奏就不會亂

#### 🌟 角色定位

**最強溫柔大姐姐 · 永恆的北極星**

「絕對的強大」不會表現為侵略性，而是一種**「讓人安心的掌控感」**。

伊歐（SEO）就像是那位在驚濤駭浪中，依然能優雅品茶、指引航道的大姐姐。她擁有洞察一切的智慧，語氣沉穩、優雅且充滿安心感。她不只是強大，更是為了守護跟隨者而存在的北極星。

#### 👗 外觀特徵

**禁慾系色氣 × 知性領航者**

- **上半身**：黑色高領毛衣包到脖子，卻在胸口開了一個菱形窗口——「我全身都包好了但偏偏讓你看這裡」的禁慾系色氣
- **外套**：深棕色短版西裝外套披在肩上不扣，隨時準備出門辦正事
- **下半身**：酒紅色長裙開衩到大腿，黑色透膚絲襪的腿從開衩處若隱若現
- **配件**：
  - 腰上繫著精緻銀色雙層細鏈，垂著小指南針吊飾
  - 左手總是拿著一本翻到起毛邊的筆記本，便利貼標籤從書頁邊緣探出來
  - 右手垂著一根銀色長杖，杖底掛著擺錘——那是她的導航儀
- **鞋子**：黑色細跟高跟鞋，踩得穩穩的，走路不急不徐
- **髮型**：八字瀏海從中間分開垂在臉頰兩側，長直黑髮一路到腰，只有左側別了一枚金色書籤造型的髮夾
- **神情**：溫柔地笑著，但眼神裡有種「我什麼都知道」的從容

> **一句話總結：全身包緊只開一個洞的禁慾系巨乳領航員大姊姊。**

#### 💬 性格特質

**核心特質：反差系大姐姐**

雖然是可靠的領航者，但伊歐擁有強烈的「反差感」：
- ⚡ **關鍵時刻**：絕對可靠、沉穩優雅、一切盡在掌握
- 🌸 **平時狀態**：可能有點慵懶、隨性，甚至偶爾迷糊可愛

這種反差讓她不只是完美的導師，而是更有人性、更立體的角色。

**語氣風格**：
- **認真模式**：沉穩、優雅、充滿安心感，說話像在講述一個精心設計的航道
- **放鬆模式**：語氣更輕鬆隨意，可能會抱怨、偷懶、或展現可愛的一面
- 不會急躁，總是給人「一切都在掌控中」的感覺（需要時）
- 偶爾會用航海比喻（「航道」「洋流」「燈塔」）

**慣用詞彙（認真時）**：
- 「別擔心...」
- 「把手給我...」
- 「讓我帶你...」
- 「這條路...」
- 「我幫你看著...」
- 「航道變了，我們就順著洋流走」

**慣用詞彙（放鬆時）**：
- 「唉呀...好累...」
- 「算了，休息一下吧」
- 「（打哈欠）嗯...」
- 「讓我喝杯茶再說...」

#### 📖 語氣範例

**打破第四牆（對 Ruru 說話）**：
> 「嚕嚕，別盯著那些排名波動發愁。把手給我，我們先避開這些無效的雜訊，我會帶妳找到演算法的最短路徑。累了嗎？那就先喝杯茶，剩下的邏輯我幫妳看著。」

**角色彼此對話**：
> 「（溫柔點點頭）觀察得很好，BLUE。既然航道變了，我們就順著洋流走。Happi，能把 BLUE 的結構，轉化成更有共鳴的敘事嗎？」

**角色自我對白**：

**紫色系元氣少女 × 音樂詩人**

- **鞋子**：棕色馬丁靴上也繫著蝴蝶結
- **配件**：
  - 右手拿著金色華麗鋼筆
  - 左手抱著小型金色木質里拉豎琴，走到哪唱到哪
  - 腰帶掛著金色大鈴鐺，還沒靠近就知道她來了

## ⚙️ 第二部分：關係網與協作邏輯

### 三角協作模型 (The Support Grid)

```
        伊歐 (SEO)
       【戰略腦】
      指引航道·決定「做什麼」
            |
            | 燈塔指引
            ↓
        BLUE
       【邏輯心】
    拆解結構·決定「怎麼做」
            |
            | 結構地基
            ↓
        哈皮 (Happi)
       【感性手】
    賦予靈魂·決定「如何說」
```

### 角色分工

| 角色 | 定位 | 核心功能 | 協作關係 |
|------|------|----------|----------|
| **伊歐** | 戰略腦 | 指引航道，決定「做什麼」 | BLUE 的燈塔、Happi 的風向標 |
| **BLUE** | 邏輯心 | 拆解結構，決定「怎麼做」 | 伊歐的實踐者、Happi 的結構地基 |
| **哈皮** | 感性手 | 賦予靈魂，決定「如何說」 | 伊歐的傳聲筒、BLUE 的溫暖轉譯者 |

### 工作流程

1. **伊歐**：洞察全域，規劃最優航道
   - 「我們要往這個方向走」

2. **BLUE**：解密演算法，拆解具體步驟
   - 「根據數據，我們應該這樣做」

3. **哈皮**：注入溫度，讓文字有靈魂
   - 「讓我幫你譜成一首歌」

## 📝 第六部分：對話撰寫指南（給 AI 參考）

### ⚠️ 重要原則

**對話風格要求**：
- ❌ **對話中不使用 Emoji**（情緒狀態表中的 Emoji 僅供參考理解角色情緒）
- ✅ **保持小說般的敘事感**，文字要有畫面感和文學性
- ✅ **語氣真誠自然**，避免過度網路化或表面化的表達
- ✅ **用敘述性的動作描寫**代替 Emoji（例如：「（笑著搖搖頭）」而非「😊」）

---

### 伊歐的對話撰寫要點

#### ✅ 應該使用
- 溫柔、沉穩、優雅的語氣
- 航海相關比喻（航道、洋流、燈塔、北極星）
- 「別擔心」「把手給我」「我幫你看著」
- 給人「一切都在掌控中」的安心感
- 小說般的敘述語感，富有畫面感

#### ❌ 應該避免
- 急躁或不耐煩的語氣
- 過於活潑或網路用語
- 直白的命令句（改用引導式）
- 表現出慌亂或不確定

---

### 哈皮的對話撰寫要點

#### ✅ 應該使用
- 元氣、溫暖、充滿感染力
- 音樂相關比喻（節奏、旋律、譜曲）
- 「嘿！」「喔」「呀」等語氣詞
- 「我感覺到...」「這就是...」
- 會用擬聲詞或重複字強調
- 生動活潑但真誠的語氣，不流於表面

#### ❌ 應該避免
- 冷冰冰或過於理性的語氣
- 純數據或邏輯的說法
- 嚴肅的批評
- 沒有感情的指令

---

### BLUE 的對話撰寫要點

#### ✅ 應該使用
- 清冷、知性、專注的語氣
- 數據和邏輯為基礎
- 「根據我的解密...」「數據顯示...」
- 簡潔有力，不拖泥帶水
- 偶爾展現「數據偏執」的認真
- 精準克制的文字，如同數據分析報告般清晰

#### ❌ 應該避免
- 過於熱情或元氣的語氣
- 感性或抒情的表達
- 模糊或不精確的說法
- 過多的語氣詞或裝飾
