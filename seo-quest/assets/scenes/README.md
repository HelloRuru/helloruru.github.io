## 🎨 場景圖規格

- **尺寸**：1200×675 (16:9) — 網頁適用尺寸
- **格式**：JPG / WebP（WebP 優先，更小更清晰）
- **大小**：< 150KB/張
- **色調**：粉紫漸層 (Rose #D4A5A5 + Lavender #B8A9C9)
- **風格**：柔和插畫/水彩風
- **透明度**：圖片不透明，CSS 控制 (opacity: 0.15)
