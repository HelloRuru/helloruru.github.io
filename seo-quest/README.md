# SEO Quest — SEO 文案學習系統

> 透過 RPG 遊戲化學習，掌握 SEO 文案撰寫技巧
> 部署於：`lab.helloruru.com/seo-quest/`

---

## 🎯 專案簡介

**SEO Quest** 是一個創新的 SEO 教學工具，結合 RPG 遊戲元素與專業工具模式，讓學習 SEO 文案變得有趣又有效。

### 核心特色

- **🎮 雙模式系統**
  - **教學模式**：RPG 風格的互動式學習體驗
  - **工具模式**：專業 SEO 文案生成與分析工具

- **📚 五階段學習循環**
  ```
  教學 → 示範 → 實作 → 評分 → 升級
  ```

- **🎯 完整評分系統**
  - 關鍵字優化（30%）
  - 關鍵字使用（30%）
  - 內容品質（40%）

- **🏆 成就與獎勵**
  - 等級系統（Lv.1 → Lv.10）
  - 解鎖新工具和筆記
  - 收集成就徽章

## 🎮 使用方式

### 教學模式

1. **選擇關卡**
   - 從關卡地圖選擇想學習的主題
   - 必須依序完成，解鎖下一關

2. **五階段學習**
   ```
   Phase 1: Tutorial（教學）
   ↓ 學習 SEO 概念和理論

   Phase 2: Demo（示範）
   ↓ 觀看 NPC 實際示範

   Phase 3: Practice（實作）
   ↓ 自己動手寫文案

   Phase 4: Score（評分）
   ↓ 獲得詳細評分與建議

   Phase 5: Levelup（升級）
   ↓ 獲得經驗值、解鎖工具
   ```

3. **追蹤進度**
   - 查看等級和經驗值
   - 收集成就徽章
   - 解鎖新工具和筆記

### 工具模式

1. **快速輸入**
   - 輸入主題和關鍵字
   - 快速生成 SEO 文案

2. **即時分析**
   - 關鍵字密度檢測
   - 可讀性分析
   - SEO 策略建議

3. **Fact-Check**
   - 內容準確性檢查
   - 來源引用建議

## 📊 評分系統

### 評分維度

| 維度 | 權重 | 說明 |
|------|------|------|
| 關鍵字選擇 | 30% | 主要和次要關鍵字的選擇是否恰當 |
| 關鍵字使用 | 30% | 密度、分布、自然度 |
| 內容品質 | 40% | 吸引力、結構、可讀性 |

## 🛠️ 開發指南

### 新增關卡

```bash
# 1. 複製模板
cp data/levels/world-1/1-1.json data/levels/world-1/1-5.json

# 2. 編輯關卡內容
# 修改 id, title, phases 等欄位

# 3. 測試關卡
# 在瀏覽器中測試新關卡
```

### 新增評分規則

編輯 `data/scoring-rules.json`，新增評分維度：

```json
{
  "newCriteria": {
    "weight": 10,
    "checks": [...]
  }
}
```

在 `js/modules/scoring.js` 實作檢查邏輯。
