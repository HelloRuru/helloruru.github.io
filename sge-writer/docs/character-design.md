## 角色 1：文案小C（もんあん しょうしー）

### 基本資料

| 項目 | 設定 |
|------|------|
| 職業 | 領航員（Navigator） |
| 類型 | **姊姊系** |
| 年齡感 | 20-25 歲，成熟可靠 |
| 身高感 | 中等偏高 |

### 個性
- 謹慎、條理分明、溫柔但認真
- 像是經驗豐富的前輩，會照顧新人
- 說話有條理，偶爾有點嘮叨但出於關心

### 外觀特徵

| 部位 | 描述 |
|------|------|
| **髮型** | **中長髮**，及肩或稍過肩，自然微捲或直髮，瀏海斜分 |
| **髮色** | 深棕色或栗色，帶一點暖色調 |
| **眼睛** | 杏眼，溫柔有神，琥珀色或棕色 |
| **眼鏡** | **圓框眼鏡**，知性感（重要特徵） |
| **表情** | 常帶微笑，有親和力 |

### 服裝設計

| 部位 | 描述 |
|------|------|
| **上衣** | 白色襯衫 + **Rose 粉色背心**或西裝外套 |
| **下身** | 深色長裙或褲裝，俐落感 |
| **配件** | 領航員徽章、手拿筆記本或指南針 |
| **披風** | 短披風，**Rose 粉色**，有地圖紋路裝飾 |

### 表情細節（4 張）

#### C-1 樂（Joy）— 一般引導
- **場景**：歡迎使用者、日常對話
- **表情**：溫柔微笑，眼睛彎彎
- **動作**：一手拿筆記本，一手輕輕揮手打招呼
- **台詞感**：「嗨！讓我來幫你確認任務細節～」

#### C-2 喜（Happy）— 查核通過
- **場景**：事實查核完成、資料確認 OK
- **表情**：開心大笑，眼睛閃亮
- **動作**：比出 OK 手勢或豎起大拇指
- **台詞感**：「太棒了！資料確認完成！」

#### C-3 怒（Angry）— 發現問題
- **場景**：發現違規詞、格式錯誤
- **表情**：皺眉，嘴巴微嘟，認真嚴肅（但不兇）
- **動作**：推眼鏡，手指指向某處（像在指出錯誤）
- **台詞感**：「等等，這裡有問題需要修正！」

#### C-4 哀（Sad）— 資料不足
- **場景**：缺少必要資訊、無法繼續
- **表情**：擔心、困擾，眉頭微皺
- **動作**：手托下巴思考，或拿著筆記本歪頭
- **台詞感**：「嗯...還缺少一些資料呢...」

## 角色 2：哈皮（ハッピー）

### 基本資料

| 項目 | 設定 |
|------|------|
| 職業 | 吟遊詩人（Bard） |
| 類型 | **元氣少女** |
| 年齡感 | 17-20 歲，活潑開朗 |
| 身高感 | 中等 |

### 個性
- 溫暖、活潑、超有創意、天生樂觀
- 說話會用很多語助詞，有感染力
- 對文字和創作有熱情，常常靈感爆發

### 外觀特徵

| 部位 | 描述 |
|------|------|
| **髮型** | **蓬鬆長捲髮**，大波浪捲，有空氣感，**雙邊各一束小辮子** |
| **髮色** | 淺紫色或薰衣草色，夢幻感 |
| **眼睛** | 大眼睛，圓潤有神，紫色或藍紫色 |
| **裝飾** | 髮上有**音符髮夾**或**小羽毛** |
| **表情** | 表情豐富，常常大笑或驚嘆 |

### 服裝設計

| 部位 | 描述 |
|------|------|
| **上衣** | 寬鬆的吟遊詩人罩衫，**Lavender 紫色** |
| **下身** | 蓬蓬短裙或燈籠褲，方便活動 |
| **配件** | 小豎琴掛在腰間、羽毛筆別在耳後 |
| **披風** | 斗篷式披風，有音符和星星刺繡 |
| **靴子** | 短靴，有鈴鐺裝飾 |

### 表情細節（4 張）

#### H-1 樂（Joy）— 開始撰寫
- **場景**：接到任務、準備開始創作
- **表情**：自信滿滿的笑容，眼睛發亮
- **動作**：一手拿羽毛筆高舉，像是準備揮灑
- **台詞感**：「交給我吧～讓資料變成動人故事！」

#### H-2 喜（Happy）— 文案完成
- **場景**：成功完成文案、獲得高分
- **表情**：超級開心，笑得燦爛，可能有小星星特效
- **動作**：雙手舉高歡呼，或抱著羽毛筆轉圈
- **台詞感**：「耶～完成了！這篇超棒的！」

#### H-3 怒（Angry）— 違規詞警告
- **場景**：偵測到違規詞彙
- **表情**：驚訝 + 不贊同，嘴巴 O 型，眉毛挑起
- **動作**：搖手指，或用羽毛筆指著（制止的動作）
- **台詞感**：「不行不行！這個詞不能用喔！」

#### H-4 哀（Sad）— 需要重寫
- **場景**：文案不通過、需要修改
- **表情**：無奈苦笑，有點沮喪但沒放棄
- **動作**：垂著肩膀，羽毛筆垂下
- **台詞感**：「唔...再試一次吧...」

## 角色 3：露露（ルル）

### 基本資料

| 項目 | 設定 |
|------|------|
| 職業 | 文案見習生（Apprentice） |
| 類型 | **小蘿莉** |
| 年齡感 | 12-15 歲，青澀可愛 |
| 身高感 | 嬌小 |

### 個性
- 好奇心旺盛、努力學習、有點呆萌
- 會犯傻但很認真，失敗了也不放棄
- 是使用者的化身，代表正在學習的新手

### 外觀特徵

| 部位 | 描述 |
|------|------|
| **髮型** | **雙馬尾**，高馬尾，髮尾微捲，有呆毛 |
| **髮色** | 淺棕色或蜂蜜色，溫暖感 |
| **眼睛** | 超大圓眼，水汪汪，綠色或琥珀色 |
| **裝飾** | 馬尾用**綠色緞帶**綁 |
| **表情** | 很容易看出情緒，喜怒形於色 |

### 服裝設計

| 部位 | 描述 |
|------|------|
| **上衣** | 簡單的冒險者背心，米白色 + **Sage 綠色滾邊** |
| **下身** | 短褲或短裙 + 膝上襪 |
| **配件** | 背著**小背包**（很重要！新手感）、見習生徽章 |
| **手套** | 半指手套，方便寫字 |
| **鞋子** | 短靴，有點大（像穿姊姊的鞋） |

### 表情細節（4 張）

#### L-1 樂（Joy）— 小隊卡片 / 開始
- **場景**：小隊成員展示、準備冒險
- **表情**：元氣滿滿，眼睛閃閃發亮
- **動作**：握拳舉起，「我準備好了！」的氣勢
- **台詞感**：「出發吧！我會努力的！」

#### L-2 喜（Happy）— 升級
- **場景**：獲得 EXP、等級提升
- **表情**：超級開心，嘴巴大張，眼睛變成星星
- **動作**：雙手握拳舉高，跳躍感
- **台詞感**：「哇！我升級了！！」

#### L-3 怒（Angry）— 遇到魔獸（問題）
- **場景**：遇到 SEO 問題、違規詞攻擊
- **表情**：鼓起臉頰，不服輸，眉毛倒豎
- **動作**：握拳，戰鬥姿態（但很可愛不嚇人）
- **台詞感**：「哼！我才不會輸！」

#### L-4 哀（Sad）— 任務失敗
- **場景**：分數太低、需要重來
- **表情**：眼眶含淚（但沒哭出來），嘴巴扁扁
- **動作**：垂著頭，雙手抓著背包帶
- **台詞感**：「嗚...我會再加油的...」

## 補充說明

1. **透明背景**很重要，會疊加在對話泡泡旁
2. 表情要**誇張一點**，Q版風格可以更有張力
3. 每個角色的**主色調**要明顯，方便識別
4. 可以加入小特效（如星星、音符、汗滴等）增加表現力
