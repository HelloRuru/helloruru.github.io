# The Hello from Ruru | AI Lab

### 蔡依庭 (Kaoru 阿嚕) 的數位實踐場
這是一個從暢銷書作者跨足 SGE (Search Generative Experience) 最佳化領域的新篇章。在這裡，我紀錄如何利用 AI 工具產製文案，並深入探究演算法邏輯。這不只是技術的應用，更是我追尋深刻答案的軌跡。

---

## 核心研究 (Core Research)
- 中文指令邏輯 (Prompt Engineering)：研究繁體中文語境下的高精準度指令架構，建立具備溫度的在地化語感。
- SGE 最短路徑最佳化：探索生成式搜尋引擎如何抓取資訊，並調整內容在 AI 回答中的出現權重。
- 演算法語意分析：拆解 AI 邏輯，將技術應用轉化為深刻的答案軌跡。
